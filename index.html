Um algoritmo de ordenação é utilizado para organizar dados de acordo com um critério definido pelo usuário. O Bubble Sort é um algoritmo de ordenação muito simples, usado principalmente para fins educacionais. O motivo disso é a velocidade, já que o Bubble Sort é muito lento para ordenar grandes conjuntos de dados, mas é muito fácil de entender.

O Bubble Sort funciona comparando um elemento com seu antecessor e trocando suas posições se o elemento for menor. Essa ideia é repetida para cada par de elementos consecutivos em um array, até que nenhuma troca aconteça, o que significa que os elementos estão ordenados.

Complexidade de tempo: caso médio: O(n²), melhor caso: O(n), pior caso: O(n²)
